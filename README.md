## 📂 โครงสร้างภายในโปรเจกต์ (Project Structure)

โปรเจกต์นี้ได้รับการพัฒนาให้มีปุ่มรันระบบที่สะดวกสบายอยู่ที่ด้านนอกสุด (Root) ของโฟลเดอร์:

* 🚀 **`1. เปิดเว็บไซต์โรงเรียน.bat`**: ดับเบิ้ลคลิกเพื่อเริ่มต้นการรันระบบจำลองหลังบ้านในคอมพิวเตอร์ของคุณทันที (จะติดตั้งสิทธิ์ npm และเปิดหน้าต่างหลังบ้านในเบราว์เซอร์ให้โดยอัตโนมัติ)
* 📂 **`2. เปิดโฟลเดอร์โค้ดระบบ.bat`**: สำหรับเรียกเปิดโฟลเดอร์ดูไฟล์ทั้งหมดในเครื่อง

ภายในจะแบ่งเป็นโฟลเดอร์ย่อย:
* **`server-version`**: โฟลเดอร์โค้ด Express Server สำหรับรันจำลองการใช้งานหลังบ้านในเครื่องคอม
* **`static-version`**: โฟลเดอร์โค้ดหน้าเว็บแบบ Static สำหรับเชื่อมโยงส่งขึ้นเว็บจริงบน **Vercel** หรือ Netlify

## 🚀 วิธีการใช้งานระบบที่สะดวกที่สุด (Easy Workflow)

สำหรับผู้ที่ดาวน์โหลดโปรเจกต์นี้ไปใช้งานหรือแก้ไขข้อมูล สามารถใช้ขั้นตอนดังนี้ได้ทันที:

1. **เปิดเว็บจำลอง:** ดับเบิ้ลคลิกไฟล์ `1. เปิดเว็บไซต์โรงเรียน.bat` ที่ด้านนอกสุดของโปรเจกต์
2. **แก้ไขข้อมูล:** เปิดหน้าเว็บหลังบ้านที่ 🔗 `http://localhost:3000/admin` (พิมพ์ ID: `admin` / PASS: `admin`) จากนั้นกดบันทึกการตั้งค่า ปรับเปลี่ยนสี หรืออัปโหลดรูปภาพได้ตามใจชอบ
3. **ระบบบันทึกอัตโนมัติ:** เมื่อคุณกดบันทึกหรืออัปโหลดรูปภาพ โค้ดจำลองหลังบ้านจะบันทึกข้อมูลทับลงไปที่โฟลเดอร์ `static-version` ให้โดยอัตโนมัติทันที
4. **อัปขึ้นเว็บจริง:** เปิดโปรแกรม **GitHub Desktop** จากนั้นเขียนสรุปหัวข้อแก้ไของค์กรของคุณ กดปุ่ม **Commit** และกด **Push origin** เพื่ออัปโหลดขึ้น Vercel / Netlify ได้ทันทีโดยไม่ต้องทำการคัดลอกไฟล์ใดๆ ด้วยตัวเอง!

## ☁️ วิธีการดีพลอยขึ้น Vercel (Vercel Deploy)

เมื่อคุณเชื่อมโยงบัญชี GitHub เข้ากับ Vercel ตอนที่ทำการ Import โปรเจกต์เข้ามา:
* ⚠️ **สำคัญมาก:** ก่อนที่จะกดปุ่ม Deploy ให้มองหาหัวข้อ **Root Directory** แล้วกด **Edit** เพื่อระบุปลายทางไปที่โฟลเดอร์ **`static-version`** เท่านั้น (ไม่ใช่รันโฟลเดอร์นอกสุด) จากนั้นจึงกด Deploy

## 🔑 ข้อมูลการเข้าสู่ระบบสำหรับผู้ดูแลระบบ (Admin Login)

เมื่อเปิดเข้าสู่หน้าจัดการระบบหลังบ้านที่ลิงก์ `/admin` (เช่น `http://localhost:3000/admin` หรือ `https://your-site.vercel.app/admin`) บัญชีผู้ดูแลระบบเริ่มต้นสำหรับลงชื่อเข้าใช้คือ:

* **ชื่อผู้ใช้ (Username):** `admin`
* **รหัสผ่าน (Password):** `admin`

*(แนะนำให้ทำการเปลี่ยนรหัสผ่านเพื่อความปลอดภัยหลังจากเข้าสู่ระบบครั้งแรกเรียบร้อยแล้ว ในแถบเมนู "เปลี่ยนรหัสผ่าน" ด้านซ้ายมือ)*

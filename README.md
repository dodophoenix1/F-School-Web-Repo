# 🏫 ระบบเว็บไซต์โรงเรียนพร้อมระบบหลังบ้าน (School Website with CMS Admin Panel)

ระบบเว็บไซต์สำเร็จรูปสำหรับโรงเรียนหรือหน่วยงานทางการศึกษา พัฒนาด้วยเทคโนโลยีเว็บสมัยใหม่ มาพร้อมระบบจัดการหลังบ้านครบวงจรและฟังก์ชันการเชื่อมต่อกับ Google Drive อย่างเต็มรูปแบบ

---

## 🌟 คุณสมบัติหลักของระบบ (Key Features)

* ✨ **Premium Aesthetic & Layout**: หน้าตาดีไซน์ทันสมัย ตอบสนองดีเยี่ยมทุกอุปกรณ์ (Responsive Layout) ตกแต่งอย่างสวยงามด้วยสีกราเดียนต์ เอฟเฟกต์เบลอหลัง (Glassmorphism) และฟอนต์ภาษาไทยยอดนิยม
* 🎨 **Theme & Brand Customization**: ปรับแต่งสีประจำสถาบันได้ 3 ค่าสี (สีหลัก, สีรอง/เน้น, สีพื้นหลัง) และเปิดส่วนตั้งค่าระยะเว้นแถบ ขนาดข้อความ และสีฟอนต์หัวเว็บได้โดยอิสระ
* 🔄 **Auto-Sync Local to Static**: แก้ไขข้อมูลและเปลี่ยนโลโก้บนระบบ Localhost หน้าต่างแอดมินจะแปลงและคัดลอกไฟล์ลงโฟลเดอร์พร้อมอัปโหลดขึ้นเว็บจริงให้โดยอัตโนมัติ
* 📁 **Google Drive Integration**:
  * **ดึงแบนเนอร์บริการ:** คัดลอกลิงก์แชร์รูปแบนเนอร์จาก Google Drive มาวาง ระบบจะสตรีมแสดงผลแบนเนอร์ออนไลน์โดยไม่เปลืองเนื้อหาเว็บ
  * **ดึงสไลด์จดหมายข่าว:** ระบุเพียงลิงก์แชร์โฟลเดอร์ Google Drive โค้ดหลังบ้านจะแอบแกะภาพจดหมายข่าวล่าสุด 10 ฉบับจาก Drive ของคุณมาแปลงเป็นสไลด์โชว์แนวตั้งให้ทันที (ไม่ต้องอัปโหลดขึ้น Vercel เอง)
* 💼 **CMS Admin Dashboard**: แผงควบคุมแบ่งสัดส่วนการแก้ไขอย่างชัดเจน ครอบคลุมถึง:
  * จัดการแถบวิ่งประกาศด่วน (Hot Info Running Marquee)
  * จัดการลิงก์เมนูหลักนำทาง (Main Navigations)
  * จัดการแบนเนอร์ลิงก์ไอคอนระบบงานภายใน/ภายนอก (Service Banners)
  * จัดการสไลด์กิจกรรมใหญ่หน้าแรก (Hero Carousel Slider)
  * จัดการบอร์ดประกาศข่าวสาร (Announcements) แนบเอกสาร PDF, Word, Excel ได้ในตัว
  * จัดการทำเนียบผู้บริหารเรียงระดับยศ (Executive Board)
  * จัดการวิดีโอนำเสนอจาก YouTube และแกลเลอรี่กิจกรรมเด่น (Activities Block)
  * จัดการภาคีพันธมิตรเครือข่ายการศึกษา (Educational Partners)
  * ระบบเปลี่ยนเนื้อหาเอกสารหน้าย่อย (Static Subpages) เช่น ประวัติ, วิสัยทัศน์, แผนที่ ฯลฯ

## 📂 โครงสร้างภายในโปรเจกต์ (Project Structure)

โปรเจกต์นี้ได้รับการพัฒนาให้มีปุ่มรันระบบที่สะดวกสบายอยู่ที่ด้านนอกสุด (Root) ของโฟลเดอร์:

* 🚀 **`1. เปิดเว็บไซต์โรงเรียน.bat`**: ดับเบิ้ลคลิกเพื่อเริ่มต้นการรันระบบจำลองหลังบ้านในคอมพิวเตอร์ของคุณทันที (จะติดตั้งสิทธิ์ npm และเปิดหน้าต่างหลังบ้านในเบราว์เซอร์ให้โดยอัตโนมัติ)
* 📂 **`2. เปิดโฟลเดอร์โค้ดระบบ.bat`**: สำหรับเรียกเปิดโฟลเดอร์ดูไฟล์ทั้งหมดในเครื่อง

ภายในจะแบ่งเป็นโฟลเดอร์ย่อย:
* **`server-version`**: โค้ดระบบ Express Server สำหรับรันจำลองการใช้งานหลังบ้านในเครื่องคอม
* **`static-version`**: โค้ดหน้าเว็บแบบ Static สำหรับเชื่อมโยงส่งขึ้นเว็บจริงบน **Vercel** หรือ Netlify

## 🛠️ ความต้องการของระบบ (Prerequisites)

ก่อนเริ่มต้นการรันจำลองระบบหลังบ้านในเครื่องตนเองหรือนำไปพัฒนาต่อ คุณจำเป็นต้องติดตั้งโปรแกรมดังนี้:

* **Node.js** (เวอร์ชัน 16.x หรือ 18.x ขึ้นไปเป็นรุ่นแนะนำ)
* **npm** (เป็นเครื่องมือจัดการชุดสิทธิ์ ซึ่งจะติดตั้งมาพร้อมกับ Node.js โดยอัตโนมัติ)
* **Web Browser** รุ่นปัจจุบันที่รองรับ HTML5 (เช่น Google Chrome, Microsoft Edge, Safari, Firefox)
* **GitHub Account** และ **Vercel / Netlify Account** (หากต้องการนำเว็บไปเปิดตัวขึ้นคลาวด์ออนไลน์ฟรี)

## 🚀 ขั้นตอนการติดตั้งและการใช้งาน (Installation & Setup)

### 1. วิธีรันระบบจำลองหลังบ้านในเครื่องตนเอง (Local Installation)

#### วิธีที่ง่ายที่สุด (ผ่าน Windows Launcher):
1. เข้าไปที่โฟลเดอร์หลักของโปรเจกต์ `F-School-Web-Repo`
2. ดับเบิ้ลคลิกไฟล์ **`1. เปิดเว็บไซต์โรงเรียน.bat`**
3. ระบบจะตรวจสอบโฟลเดอร์สิทธิ์ และดาวน์โหลดโปรแกรมประกอบให้อัตโนมัติ (เฉพาะครั้งแรก) จากนั้นจะทำการรันเซิร์ฟเวอร์จำลองและเปิดหน้าเว็บแอดมินให้คุณในทันทีที่ลิงก์ 🔗 `http://localhost:3000/admin`

#### วิธีเปิดรันเซิร์ฟเวอร์ด้วยตนเองผ่าน Terminal:
1. เปิด Command Prompt หรือ PowerShell แล้วสลับพาธเข้าไปที่โฟลเดอร์ `server-version`
   ```bash
   cd server-version
   ```
2. พิมพ์คำสั่งเพื่อดาวน์โหลดโมดูลและสิทธิ์การรัน (ทำครั้งแรกครั้งเดียว):
   ```bash
   npm install
   ```
3. เริ่มต้นรันเซิร์ฟเวอร์ Express:
   ```bash
   npm start
   ```
4. เปิดบราวเซอร์แล้วไปที่ 🔗 `http://localhost:3000/admin`

---

### 2. วิธีการนำขึ้นเว็บจริงออนไลน์ฟรีผ่าน Vercel (Vercel Deploy Step)

1. ทำการ Push หรือเชื่อมต่อบัญชี GitHub ของคุณ
2. นำโฟลเดอร์โปรเจกต์นี้อัปโหลดขึ้นบัญชี GitHub ของคุณ
3. เปิดหน้าต่างบัญชี **Vercel** จากนั้นเลือกกดปุ่ม **Import Project** จาก Repository ที่สร้างขึ้น
4. ⚠️ **ขั้นตอนสำคัญมาก:** ก่อนที่จะกดปุ่ม Deploy ให้ไปที่หัวข้อ **Root Directory** จากนั้นเลือกกดปุ่ม **Edit** เพื่อระบุตำแหน่งโฟลเดอร์หลักไปที่โฟลเดอร์ **`static-version`** เท่านั้น (ไม่ใช่รันโฟลเดอร์นอกสุด)
5. กดปุ่มสีขาว **Deploy** เพื่อเปิดระบบออนไลน์ฟรีได้ทันที!

## 🔑 ข้อมูลการเข้าสู่ระบบสำหรับผู้ดูแลระบบ (Admin Login)

เมื่อเปิดเข้าสู่หน้าจัดการระบบหลังบ้านที่ลิงก์ `/admin` (เช่น `http://localhost:3000/admin` หรือ `https://your-site.vercel.app/admin`) บัญชีผู้ดูแลระบบเริ่มต้นสำหรับลงชื่อเข้าใช้คือ:

* **ชื่อผู้ใช้ (Username):** `admin`
* **รหัสผ่าน (Password):** `admin`

*(แนะนำให้ทำการเปลี่ยนรหัสผ่านเพื่อความปลอดภัยหลังจากเข้าสู่ระบบครั้งแรกเรียบร้อยแล้ว ในแถบเมนู "เปลี่ยนรหัสผ่าน" ด้านซ้ายมือ)*

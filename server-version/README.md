# 🏫 เทมเพลตเว็บไซต์โรงเรียนสำเร็จรูป พร้อมระบบหลังบ้าน (School Website Template & Admin Dashboard)

เทมเพลตเว็บไซต์พอร์ทัลโรงเรียนแบบไทย 3 คอลัมน์ ดีไซน์สวยงามระดับพรีเมียมด้วยสไตล์ Glassmorphic และ Responsive (รองรับมือถือ) มาพร้อมระบบหลังบ้าน (Admin Panel) ในตัวสำหรับแก้ไขข้อมูลโรงเรียน เผยแพร่ข่าวสาร จัดการแบนเนอร์เชื่อมโยงบริการ และบอร์ดผู้บริหารได้อย่างง่ายดาย 

**เหมาะสำหรับ:** นำไปพัฒนาต่อยอด แจกจ่ายฟรี หรือใช้เป็นเว็บเริ่มต้นของโรงเรียนขนาดเล็กถึงขนาดใหญ่ โดยไม่ต้องเตรียมระบบฐานข้อมูล SQL ให้ยุ่งยาก เพราะใช้ระบบเก็บข้อมูลแบบ JSON File-Based Database

---

## ✨ คุณสมบัติเด่น (Features)

1. **ระบบหลังบ้านในตัว (Admin Panel Dashboard):**
   - **จัดการข้อมูลทั่วไป (General Configuration):** เปลี่ยนชื่อโรงเรียน (ไทย/อังกฤษ), โลโก้, คำขวัญ, ข้อมูลการติดต่อ, และแก้ไขสีประจำโรงเรียน (สีหลัก/สีรอง/สีพื้นหลัง) ได้ทันทีผ่านหน้าต่างเลือกสี (Color Picker)
   - **จัดการลิงก์เมนูนำทาง (Navigation Links):** เพิ่ม ลบ แก้ไข ลิงก์เมนูด้านซ้ายของเว็บ ทั้งลิงก์ภายในและภายนอก
   - **จัดการแบนเนอร์บริการ (Banners Link):** จัดการลิงก์เชื่อมต่อบริการด่วน เช่น ระบบเกรด SGS, ระบบส่งแผนการสอน, ลิงก์ Drive ฯลฯ
   - **จัดการสไลด์หน้าแรก (Hero Slider):** อัปโหลดรูปภาพสไลด์ประชาสัมพันธ์ ใส่คำบรรยาย และลิงก์ปลายทาง
   - **เขียนข่าวสาร & ประกาศโรงเรียน (Announcements):** เขียนข่าวสารแยกตามหมวดหมู่ อัปโหลดภาพปก และอัปโหลดไฟล์เอกสารแนบ (PDF, Word, Excel) เพื่อให้ผู้ใช้งานดาวน์โหลดได้ทันที
   - **จัดการทำเนียบผู้บริหาร (Management Board):** จัดทำทำเนียบรูปถ่ายผู้บริหารพร้อมยศ ตำแหน่ง และจัดเรียงลำดับการขึ้นแสดงผล
   - **ภาพกิจกรรมเด่น (Activity Gallery):** ลงรูปกิจกรรมโรงเรียนสั้นๆ พร้อมระบบขยายรูปภาพ (Lightbox Modal) บนหน้าเว็บจริง
   - **จัดการหน้าเว็บย่อย (Static Pages):** แก้ไขบทความ ประวัติความเป็นมา, วิสัยทัศน์/พันธกิจ, และตราสัญลักษณ์ของโรงเรียน ได้แบบไดนามิกโดยตรงจากหลังบ้าน ไม่ต้องแก้ไขโค้ด HTML

2. **ดีไซน์และการแสดงผล (Premium UI/UX):**
   - ตกแต่งสไตล์ **Glassmorphism** ผสมความโปร่งใสและเงาสะท้อนที่ดูหรูหราพรีเมียม
   - จัดวางเลย์เอาต์ **3 คอลัมน์** ตามมาตรฐานเว็บโรงเรียนของไทย
   - ระบบ **Dynamic Theme Coloring** โทนสีหลักและปุ่มต่างๆ บนหน้าเว็บจะเปลี่ยนไปตามโทนสีโรงเรียนที่คุณเลือกจากห้องแอดมินทันที
   - **Responsive Design** แสดงผลได้ดีเยี่ยมบนมือถือ แท็บเล็ต และคอมพิวเตอร์
   - **Image Fallback System:** หากระบบไม่พบรูปที่อัปโหลด จะทำการสร้างภาพ SVG จำลองขึ้นมาแสดงผลโดยอัตโนมัติ เพื่อป้องกันปัญหารูปภาพแตก/ไม่แสดงผล

3. **ความปลอดภัยสูง (Security Built-in):**
   - **ป้องกัน Path Traversal:** ปิดกั้นการส่ง path พารามิเตอร์แบบแปลกปลอม (เช่น `../`) ในการอัปโหลดหรือลบไฟล์ เพื่อปกป้องไฟล์ระบบอื่นๆ ในโฮสติ้ง
   - **จำกัดสิทธิ์อัปโหลดไฟล์:** อนุญาตเฉพาะรูปภาพและประเภทไฟล์เอกสารที่ปลอดภัยเท่านั้น
   - **ระบบ Authentication:** ระบบป้องกันการบันทึกข้อมูลผ่านการตรวจสอบ Token-based บนเซิร์ฟเวอร์ก่อนทำการเขียนลงฐานข้อมูล JSON ทุกครั้ง

## 🚀 วิธีการติดตั้งและรันใช้งานภายในเครื่อง (Local Setup)

### ความต้องการของระบบ (Prerequisites)
- ติดตั้ง [Node.js](https://nodejs.org/) (เวอร์ชัน 18 ขึ้นไปแนะนำ) ในคอมพิวเตอร์ของคุณ

### ขั้นตอนการเริ่มใช้งานแบบเร็ว (สำหรับ Windows)
1. ดับเบิลคลิกไฟล์ **`run-server.bat`**
2. ระบบจะทำการตรวจสอบและดาวน์โหลดปลั๊กอิน (dependencies) ที่ต้องใช้อัตโนมัติในครั้งแรก
3. หน้าต่างเบราว์เซอร์จะเปิดเข้าสู่หน้าเว็บโรงเรียนโดยตรงที่ลิงก์ `http://localhost:3000`

### การสั่งรันผ่าน Command Line (ทุกระบบปฏิบัติการ)
1. เปิด Command Prompt หรือ Terminal ในโฟลเดอร์โครงการ
2. รันคำสั่งติดตั้ง dependencies:
   ```bash
   npm install
   ```
3. สั่งรันเซิร์ฟเวอร์:
   ```bash
   npm start
   ```
4. เปิดเบราว์เซอร์แล้วพิมพ์: `http://localhost:3000`
